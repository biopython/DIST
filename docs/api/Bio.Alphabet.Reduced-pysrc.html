  8  """Reduced alphabets which lump together several amino-acids into one letter. 
  9   
 10  Reduced (redundant or simplified) alphabets are used to represent protein 
 11  sequences using an alternative alphabet which lumps together several 
 12  amino-acids into one letter, based on physico-chemical traits. For example, 
 13  all the aliphatics (I,L,V) are usually quite interchangeable, so many sequence 
 14  studies group them into one letter 
 15   
 16  Examples of reduced alphabets are available in: 
 17   
 18  http://viscose.herokuapp.com/html/alphabets.html 
 19   
 20  The Murphy tables are from here: 
 21   
 22  Murphy L.R., Wallqvist A, Levy RM. (2000) Simplified amino acid 
 23  alphabets for protein fold recognition and implications for folding. 
 24  Protein Eng. 13(3):149-152 
 25   
 26  These alphabets have been used with Bio.utils.reduce_sequence, which has been 
 27  removed from Biopython. You can use this is alphabets and tables like this: 
 28   
 29      >>> from Bio.Seq import Seq 
 30      >>> from Bio import Alphabet 
 31      >>> from Bio.Alphabet import Reduced 
 32      >>> my_protein = Seq('MAGSKEWKRFCELTINEA', Alphabet.ProteinAlphabet()) 
 33   
 34  Now, we convert this sequence into a sequence which only recognizes polar (P) 
 35  or hydrophobic (H) residues: 
 36   
 37      >>> new_protein = Seq('', Alphabet.Reduced.HPModel()) 
 38      >>> for aa in my_protein: 
 39      ...     new_protein += Alphabet.Reduced.hp_model_tab[aa] 
 40      >>> new_protein 
 41      Seq('HPPPPPHPPHHPHPHPPP', HPModel()) 
 42   
 43  The following Alphabet classes are available: 
 44   
 45   - Murphy15: Maps 20 amino acids to 15, use murphy_15_tab for conversion, 
 46               ambiguous letters: L: LVIM, F: FY, K: KR 
 47   - Murphy10: Maps 20 amino acids to 10, use murphy_10_tab for conversion, 
 48               ambiguous letters: L: LVIM, S: ST, F: FYW, E: EDNQ, K: KR 
 49   - Murphy8: Maps 20 amino acids to 8, use murphy_8_tab for conversion, 
 50              ambiguous letters: L: LVIMC, A: AG, S: ST, F: FYW, E: EDNQ, 
 51              K: KR 
 52   - Murphy4: Maps 20 amino acids to 4, use murphy_4_tab for conversion, 
 53              ambiguous letters: L: LVIMC, A: AGSTP, F: FYW, E: EDNQKRH 
 54   - HPModel: Groups amino acids as polar (hydrophilic) or hydrophobic 
 55              (non-polar), use hp_model_tab for conversion, 
 56              P: AGTSNQDEHRKP, H: CMFILVWY 
 57   - PC5: Amino acids grouped according to 5 physico-chemical properties, 
 58          use pc_5_table for conversion, 
 59          A (Aliphatic): IVL, R (aRomatic): FYWH, C (Charged): KRDE, T (Tiny): 
 60          GACS, D (Diverse): TMQNP 
 61  """
