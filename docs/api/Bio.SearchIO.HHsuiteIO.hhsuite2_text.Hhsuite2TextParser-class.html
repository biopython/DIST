_process_score_line(line, hit_data)
Static Method

source code 

Parse the scores from the line and populate hit_data dict (PRIVATE).

Lines are of the form: Probab=99.95 E-value=3.7e-34 Score=210.31 Aligned_cols=171 Identities=100% Similarity=2.050 Sum_probs=166.9

E-value could be in decimal or scientific notation, so split the string rather then use regexp - this also means we should be tolerant of additional fields being added/removed
