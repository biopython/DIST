Module Vector

source code

Deprecated in favor of Bio.PDB.vectors to solve name collision.

Historically the meaning of Bio.PDB.Vector has been ambiguous, both a class Vector and the module which defines this class and related functions (Bio/PDB/Vector.py).

The module has been renamed to Bio.PDB.vectors (lower case in line with PEP8 guidelines).

Please use the following style imports in order to work on both old and new versions of Biopython:

>>> from Bio.PDB import calc_angle
>>> from Bio.PDB import calc_dihedral
>>> from Bio.PDB import m2rotaxis
>>> from Bio.PDB import refmat
>>> from Bio.PDB import rotaxis2m
>>> from Bio.PDB import rotmat
>>> from Bio.PDB import vector_to_axis
>>> from Bio.PDB import Vector  # for the class
