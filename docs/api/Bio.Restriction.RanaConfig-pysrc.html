 9  ############################################################################### 
10  #                   Configuration of the console. 
11  # 
12  #   Mainly used  by PrintFormat.PrintFormat 
13  # 
14  #   ConsoleWidth : width of the console used default to 80. 
15  #                   should never be less than 60. 
16  #   NameWidth    : space attributed to the name in PrintList method. 
17  #   Indent       : Indent of the second line. 
18  #   MaxSize      : Maximal size of the sequence (default=6: 
19  #                                                -> 99 999 bp + 1 trailing ',' 
20  #                  people are unlikely to ask for restriction map of sequences 
21  #                  bigger than 100.000 bp. This is needed to determine the 
22  #                  space to be reserved for sites location. 
23  # 
24  #                  MaxSize = 5  =>   9.999 bp 
25  #                  MaxSize = 6  =>  99.999 bp 
26  #                  MaxSize = 7  => 999.999 bp 
27  #   example: 
28  # 
29  #   <------------ ConsoleWidth ---------------> 
30  #   <- NameWidth -> 
31  #   EcoRI         :   1, 45, 50, 300, 400, 650, 
32  #                         700, 1200, 2500. 
33  #                     <--> 
34  #                       Indent 
35  # 
36  ConsoleWidth = 80 
37  NameWidth = 10 
38  Indent = 4 
39  MaxSize = 6
