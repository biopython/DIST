 10  """Provides code to access NCBI over the WWW. 
 11   
 12  The main Entrez web page is available at: 
 13  http://www.ncbi.nlm.nih.gov/Entrez/ 
 14   
 15  Entrez Programming Utilities web page is available at: 
 16  http://www.ncbi.nlm.nih.gov/books/NBK25501/ 
 17   
 18  This module provides a number of functions like ``efetch`` (short for 
 19  Entrez Fetch) which will return the data as a handle object. This is 
 20  a standard interface used in Python for reading data from a file, or 
 21  in this case a remote network connection, and provides methods like 
 22  ``.read()`` or offers iteration over the contents line by line. See 
 23  also "What the heck is a handle?" in the Biopython Tutorial and 
 24  Cookbook: http://biopython.org/DIST/docs/tutorial/Tutorial.html 
 25  http://biopython.org/DIST/docs/tutorial/Tutorial.pdf 
 26   
 27  Unlike a handle to a file on disk from the ``open(filename)`` function, 
 28  which has a ``.name`` attribute giving the filename, the handles from 
 29  ``Bio.Entrez`` all have a ``.url`` attribute instead giving the URL 
 30  used to connect to the NCBI Entrez API. 
 31   
 32  The Entrez module also provides an XML parser which takes a handle 
 33  as input. 
 34   
 35  Variables: 
 36   
 37      - email        Set the Entrez email parameter (default is not set). 
 38      - tool         Set the Entrez tool parameter (default is ``biopython``). 
 39      - api_key      Personal API key from NCBI. If not set, only 3 queries per 
 40                     seconds are allowed. 10 queries per seconds otherwise with a 
 41                     valid API key. 
 42   
 43  Functions: 
 44   
 45      - efetch       Retrieves records in the requested format from a list of one or 
 46        more primary IDs or from the user's environment 
 47      - epost        Posts a file containing a list of primary IDs for future use in 
 48        the user's environment to use with subsequent search strategies 
 49      - esearch      Searches and retrieves primary IDs (for use in EFetch, ELink, 
 50        and ESummary) and term translations and optionally retains 
 51        results for future use in the user's environment. 
 52      - elink        Checks for the existence of an external or Related Articles link 
 53        from a list of one or more primary IDs.  Retrieves primary IDs 
 54        and relevancy scores for links to Entrez databases or Related 
 55        Articles;  creates a hyperlink to the primary LinkOut provider 
 56        for a specific ID and database, or lists LinkOut URLs 
 57        and Attributes for multiple IDs. 
 58      - einfo        Provides field index term counts, last update, and available 
 59        links for each database. 
 60      - esummary     Retrieves document summaries from a list of primary IDs or from 
 61        the user's environment. 
 62      - egquery      Provides Entrez database counts in XML for a single search 
 63        using Global Query. 
 64      - espell       Retrieves spelling suggestions. 
 65      - ecitmatch    Retrieves PubMed IDs (PMIDs) that correspond to a set of 
 66        input citation strings. 
 67   
 68      - read         Parses the XML results returned by any of the above functions. 
 69        Typical usage is: 
 70   
 71            >>> from Bio import Entrez 
 72            >>> Entrez.email = "Your.Name.Here@example.org" 
 73            >>> handle = Entrez.einfo() # or esearch, efetch, ... 
 74            >>> record = Entrez.read(handle) 
 75            >>> handle.close() 
 76   
 77         where record is now a Python dictionary or list. 
 78   
 79      - parse        Parses the XML results returned by those of the above functions 
 80        which can return multiple records - such as efetch, esummary 
 81        and elink. Typical usage is: 
 82   
 83            >>> handle = Entrez.esummary(db="pubmed", id="19304878,14630660", retmode="xml") 
 84            >>> records = Entrez.parse(handle) 
 85            >>> for record in records: 
 86            ...     # each record is a Python dictionary or list. 
 87            ...     print(record['Title']) 
 88            Biopython: freely available Python tools for computational molecular biology and bioinformatics. 
 89            PDB file parser and structure class implemented in Python. 
 90            >>> handle.close() 
 91   
 92        This function is appropriate only if the XML file contains
