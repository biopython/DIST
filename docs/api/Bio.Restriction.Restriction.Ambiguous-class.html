Implement the methods specific to the enzymes for which the overhang is variable.

Typical example : BstXI -> CCAN_NNNN^NTGG
The overhang can be any sequence of 4 bases.
Notes:
Blunt enzymes are always defined. even if there site is GGATCCNNN^_N There overhang is always the same : blunt!

Internal use only. Not meant to be instantiated.
