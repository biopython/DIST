  7  """Provides code to access the TogoWS integrated websevices of DBCLS, Japan. 
  8   
  9  This module aims to make the TogoWS (from DBCLS, Japan) easier to use. See: 
 10  http://togows.dbcls.jp/ 
 11   
 12  The TogoWS REST service provides simple access to a range of databases, acting 
 13  as a proxy to shield you from all the different provider APIs. This works using 
 14  simple URLs (which this module will construct for you). For more details, see 
 15  http://togows.dbcls.jp/site/en/rest.html 
 16   
 17  The functionality is somewhat similar to Biopython's Bio.Entrez module which 
 18  provides access to the NCBI's Entrez Utilities (E-Utils) which also covers a 
 19  wide range of databases. 
 20   
 21  Currently TogoWS does not provide any usage guidelines (unlike the NCBI whose 
 22  requirements are reasonably clear). To avoid risking overloading the service, 
 23  Biopython will only allow three calls per second. 
 24   
 25  The TogoWS SOAP service offers a more complex API for calling web services 
 26  (essentially calling remote functions) provided by DDBJ, KEGG and PDBj. For 
 27  example, this allows you to run a remote BLAST search at the DDBJ. This is 
 28  not yet covered by this module, however there are lots of Python examples 
 29  on the TogoWS website using the SOAPpy python library. See: 
 30  http://togows.dbcls.jp/site/en/soap.html 
 31  http://soapy.sourceforge.net/ 
 32  """
