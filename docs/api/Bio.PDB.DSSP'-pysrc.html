  8  You need to have a working version of DSSP (and a license, free for academic 
  9  use) in order to use this. For DSSP, see http://swift.cmbi.ru.nl/gv/dssp/.

 11  The following Accessible surface area (ASA) values can be used, defaulting 
 12  to the Sander and Rost values: 
 13   
 14      Miller 
 15          Miller et al. 1987 https://doi.org/10.1016/0022-2836(87)90038-6 
 16      Sander 
 17          Sander and Rost 1994 https://doi.org/10.1002/prot.340200303 
 18      Wilke 
 19          Tien et al. 2013 https://doi.org/10.1371/journal.pone.0080635 
 20   
 21  The DSSP codes for secondary structure used here are: 
 22   
 23      =====     ==== 
 24      Code      Structure 
 25      =====     ==== 
 26       H        Alpha helix (4-12) 
 27       B        Isolated beta-bridge residue 
 28       E        Strand 
 29       G        3-10 helix 
 30       I        Pi helix 
 31       T        Turn 
 32       S        Bend 
 33       \-       None 
 34      =====     ==== 
 35   
 36  Usage 
 37  ----- 
 38  The DSSP class can be used to run DSSP on a pdb file, and provides a 
 39  handle to the DSSP secondary structure and accessibility. 
 40   
 41  **Note** that DSSP can only handle one model, and will only run 
 42  calculations on the first model in the provided PDB file. 
 43   
 44  Examples 
 45  -------- 
 46  >>> p = PDBParser() 
 47  >>> structure = p.get_structure("1MOT", "1mot.pdb") 
 48  >>> model = structure[0] 
 49  >>> dssp = DSSP(model, "1mot.pdb") 
 50   
 51  Note that the recent DSSP executable from the DSSP-2 package was 
 52  renamed from `dssp` to `mkdssp`. If using a recent DSSP release, 
 53  you may need to provide the name of your DSSP executable: 
 54   
 55  >>> dssp = DSSP(model, '1mot.pdb', dssp='mkdssp') 
 56   
 57  DSSP data is accessed by a tuple - (chain id, residue id): 
 58   
 59  >>> a_key = list(dssp.keys())[2] 
 60  >>> a_key 
 61  ('A', (' ', 251, ' ')) 
 62  >>> dssp[a_key] 
 63  (3, 'A', 'H', 0.7075471698113207, -61.2, -42.4, 
 64   -2, -0.7, 4, -3.0, 1, -0.2, 5, -0.2) 
 65   
 66  The dssp data returned for a single residue is a tuple in the form: 
 67   
 68      ============ === 
 69      Tuple Index  Value 
 70      ============ === 
 71      0            DSSP index 
 72      1            Amino acid 
 73      2            Secondary structure 
 74      3            Relative ASA 
 75      4            Phi 
 76      5            Psi 
 77      6            NH-->O_1_relidx 
 78      7            NH-->O_1_energy 
 79      8            O-->NH_1_relidx 
 80      9            O-->NH_1_energy 
 81      10           NH-->O_2_relidx 
 82      11           NH-->O_2_energy 
 83      12           O-->NH_2_relidx 
 84      13           O-->NH_2_energy 
 85      ============ ===
