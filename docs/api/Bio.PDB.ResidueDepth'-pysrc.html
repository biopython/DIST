  7  """Calculation of residue depth using command line tool MSMS. 
  8   
  9  This module uses Michel Sanner's MSMS program for the surface calculation. 
 10  See: http://mgltools.scripps.edu/packages/MSMS 
 11   
 12  Residue depth is the average distance of the atoms of a residue from 
 13  the solvent accessible surface. 
 14   
 15  Residue Depth:: 
 16   
 17      from Bio.PDB.ResidueDepth import ResidueDepth 
 18      from Bio.PDB.PDBParser import PDBParser 
 19      parser = PDBParser() 
 20      structure = parser.get_structure("1a8o", "Tests/PDB/1A8O.pdb") 
 21      model = structure[0] 
 22      rd = ResidueDepth(model) 
 23      print(rd['A',(' ', 152, ' ')]) 
 24   
 25  Direct MSMS interface, typical use:: 
 26   
 27      from Bio.PDB.ResidueDepth import get_surface 
 28      surface = get_surface(model) 
 29   
 30  The surface is a Numeric array with all the surface vertices. 
 31   
 32  Distance to surface:: 
 33   
 34      from Bio.PDB.ResidueDepth import min_dist 
 35      coord = (1.113, 35.393,  9.268) 
 36      dist = min_dist(coord, surface) 
 37   
 38  where coord is the coord of an atom within the volume bound by 
 39  the surface (ie. atom depth). 
 40   
 41  To calculate the residue depth (average atom depth of the atoms 
 42  in a residue):: 
 43   
 44      from Bio.PDB.ResidueDepth import residue_depth 
 45      chain = model['A'] 
 46      res152 = chain[152] 
 47      rd = residue_depth(res152, surface)
