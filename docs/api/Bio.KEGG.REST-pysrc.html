  8  """Provides code to access the REST-style KEGG online API. 
  9   
 10  This module aims to make the KEGG online REST-style API easier to use. See: 
 11  http://www.kegg.jp/kegg/rest/keggapi.html 
 12   
 13  The KEGG REST-style API provides simple access to a range of KEGG databases. 
 14  This works using simple URLs (which this module will construct for you), 
 15  with any errors indicated via HTTP error levels. 
 16   
 17  The functionality is somewhat similar to Biopython's Bio.TogoWS and Bio.Entrez 
 18  modules. 
 19   
 20  Currently KEGG does not provide any usage guidelines (unlike the NCBI whose 
 21  requirements are reasonably clear). To avoid risking overloading the service, 
 22  Biopython will only allow three calls per second. 
 23   
 24  References: 
 25   
 26  Kanehisa, M. and Goto, S.; KEGG: Kyoto Encyclopedia of Genes and Genomes. 
 27  Nucleic Acids Res. 28, 29-34 (2000). 
 28  """
