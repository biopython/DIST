 16      """Count the number of items in an iterable. 
 17   
 18      If the argument supports len(items), and some iterators do, then 
 19      this returns len(items). Otherwise it will scan over the entries 
 20      in order to count them. 
 21   
 22      Exhausts a generator, but doesn't require creating a full list.

 24      >>> iterlen("abcde")

 31          # e.g. Under Python 2, the xrange iterator defines __len__ 
 32          return len(items)
