  8  """Bio.AlignIO support for GCG MSF format. 
  9   
 10  The file format was produced by the GCG PileUp and and LocalPileUp tools, 
 11  and later tools such as T-COFFEE and MUSCLE support it as an optional 
 12  output format. 
 13   
 14  The original GCG toola would write gaps at ends of each sequence which could 
 15  be missing data as tildes (``~``), whereas internal gaps were periods (``.``) 
 16  instead. This parser replaces both with minus signs (``-``) for consistency 
 17  with the rest of ``Bio.AlignIO``. 
 18   
 19  You are expected to use this module via the Bio.AlignIO functions (or the 
 20  Bio.SeqIO functions if you want to work directly with the gapped sequences).
