Module GckIO

source code

Bio.SeqIO support for the "gck" file format.

The GCK binary format is generated by the Gene Construction Kit software from Textco BioSoftware, Inc.

_read_packet(handle)

source code 

Read a length-prefixed packet.

Parts of a GCK file are made of "packets" comprising of 4 bytes giving the packet's size, followed by the packet's data.

There is no type tag. The type of a packet, and thus the type of data it contains, is solely indicated by the position of the packet within the GCK file.

source code 

Read a Pascal string.

A Pascal string is one byte for length followed by the actual string.

_read_p4string(handle)

source code 

Read a 32-bit Pascal string.

Similar to a Pascal string but length is encoded on 4 bytes.

GckIterator(handle)

source code 

Parse a GCK file and return a SeqRecord object.

Note that a GCK file can only contain one sequence, so this iterator will always return a single record.
