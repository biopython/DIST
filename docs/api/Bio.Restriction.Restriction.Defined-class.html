Implement the methods specific to the enzymes for which the overhang and the cut are not variable.

Typical example : EcoRI -> G^AATT_C
The overhang will always be AATT
Notes:
Blunt enzymes are always defined. even if there site is GGATCCNNN^_N There overhang is always the same : blunt!

Internal use only. Not meant to be instantiated.
